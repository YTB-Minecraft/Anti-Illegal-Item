## 功能
### 32k

###### 以下是防32K機制以下是防32K機制
- 所有容器裡的32k會在打開容器時變成一般武器(超疊加也會)
- 使用方法很簡單，打開漏斗丟出一個壞掉的接著他會漏掉一個32k。漏掉的可以拿走。
- 在打人的時候系統會檢測你的背包並移除32k

### 防超疊加物品
###### 以下是防32K機制以下是防超疊加物品機制
- 跟32k一樣，在打開容器時會檢測，但打人不會。
- 用漏斗漏(跟2B2T一樣)就可解決問題


*這個機制是模仿2B2T PE的最後一個有32K的版本!只不過是在打人的時候檢查!
